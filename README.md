# java-explore-with-me
Template repository for ExploreWithMe project.
https://github.com/ardivad2015/java-explore-with-me/pull/3
### Краткое описание дополнительной функциональности:

#### Пользовательский API работы с комментариями:

```java
Добавление локации администратором:
        Post        /admin/venues
        Тело запроса
        {
           "location": {
                          "lat": float,
                          "lon": float
           },
           "name": String 
        }
        
Получение списка локаций:
        Get         /admin/venues
        Параметры запроса:
        ids array[Long] id локаций
        from int Default value : 0 количество элементов, которые нужно пропустить для формирования текущего набора
        size int Default value : 10 количество элементов в наборе
        
Обновление локации:
        Patch       /admin/venues
        Параметры пути запроса:
        id Long id локации обязательный
        Тело запроса:
        {
           "location": {
                          "lat": float,
                          "lon": float
           },
           "name": String 
        }
        
Удаление локации:
        Delete       /admin/venues
        Параметры пути запроса:
        id Long id локации обязательный
       
Для поиска событий, которые проводятся в конкретной локации, в запросе для получения событий  
основного сервиса добавлены парамтеры "venue" и "radius".

         Get   /admin/events?venue=3&radius=1from=0&size=1000

При создании события пользователь может передать id локации в поле "venue", тогда событию будет назначена 
указанная локация и координаты проведения мероприятия будут скопированы из выбранной локации.

{
  "annotation": "Сплав на байдарках похож на полет.",
  "category": 2,
  "description": "Сплав на байдарках похож на полет.",
  "eventDate": "2024-12-31 15:10:05",
  "paid": true,
  "participantLimit": 10,
  "requestModeration": false,
  "title": "Сплав на байдарках",
  "venue": 4
}

При поиске событий в конкретной локации в результат будут попадать следующие события:
1. У которых указана выбранная локация.
2. У которых локация не указана, и расстояние от точки с координатами мероприятия до точки с координатами выбранной
   локации в километрах меньше, чем значение парамтера запроса "radius".
